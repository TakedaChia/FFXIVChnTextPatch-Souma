# FFXIVChnTextPatch-Souma

## 使用

- 系统设置 - 图像设置 - 界面分辨率 - 类型 必须选择"高分辨率（WQHD/4K）"
- 将“汉化文件”中的 6 个文件覆盖到“...\FINAL FANTASY XIV - A Realm Reborn\game\sqpack\ffxiv\”
- 版本更新后必须等待更新，不可使用旧版本文件
